# 🧘‍♂️ Mindset Companion 

An AI-powered philosophical guide engineered to provide calm, actionable clarity. Built to help users navigate burnout, anxiety, and loss of focus using principles of Stoicism and detachment (Nishkama Karma).

## 🚀 Architecture & Tech Stack
- **Backend:** Node.js with Express.js for robust, asynchronous routing.
- **AI Integration:** Google Gemini 2.5 Flash API with a heavily engineered system prompt to enforce a compassionate, grounded persona.
- **Frontend:** Vanilla HTML/JS with Tailwind CSS for a premium, lightweight, and responsive UI.
- **Security:** Environment variables (`dotenv`) implemented to secure API credentials.

## ✨ Core Features
- **Dynamic State Selection:** Users select their current mental state (Burned Out, Anxious, Overwhelmed), which dynamically frames the AI's contextual response.
- **Engineered Persona:** The AI bypasses standard robotic responses, acting instead as a supportive mentor focusing on actionable clarity.
- **Immersive UI:** A calming, light-theme aesthetic designed to reduce cognitive load.
